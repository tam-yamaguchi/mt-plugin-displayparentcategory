mt-plugin-displayparentcategory
===============================
Movable Typeのブログ記事一覧に親カテゴリを表示するプラグイン

使い方
------
プラグインディレクトリにアップし、管理画面より「プラグインを有効にする」にチェックを入れてご使用ください。  
version 5.14にて動作確認
